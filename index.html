  • How it Works

    We lookup on GitHub for users who primarily code in the language of your choice, using the GitHub API. Then, we geocode those locations using the Yahoo! PlaceFinder API. Finally, we use jVectorMap to visualize the collected data.

  • Be responsible

    The Yahoo! PlaceFinder API web service has a daily cap of 50,000 requests per IP. Since Yahoo! doesn't seem to provide a service for bulk geocoding at the moment and we are not caching geocoding requests (yet!) please use this service responsibly, to avoid being banned from Yahoo!'s web services. You've been warned.

  • Contribute

    The project is completely open source and waiting for you on GitHub. Feel free to fork it, improve it and send us your pull requests. Raise issues if something is not working for you or if you have a feature request. We'll be glad to help, whenever we can.
